## Description
Der Dark-Mode-Toggle ist ein Angular-Komponente, die es ermöglicht, zwischen einem hellen und einem dunklen Design zu wechseln.

## Use
```html
<app-dark-mode-toggle></app-dark-mode-toggle>
```
